## Vite
建置專案、起環境、打包工具，效率比 vue cli快非常多

## 此專案開發&建立方式
1. 安裝node.js 
2. 安裝 yarn 
3. 安裝 vite 
4. 輸入yarn create vite並執行：建立vue3專案 
5. 輸入yarn並執行：安裝專案所需模組，會跑出node_modules資料夾和yarn.lock檔案
6. 輸入yarn dev並執行：執行專案
